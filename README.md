# Proyecto: Panadería "La Joya"

Este es el proyecto final para la materia Analisis de datos y aplicaciones web.

## Propósito
El objetivo es crear un sitio web funcional para una panadería, que sirva como base para añadir funcionalidad de back-end (PHP y MySQL) más adelante.

## Plantilla Usada
Decidimos no usar una plantilla pre-hecha, sino construir nuestra propia estructura HTML5 y CSS3 desde cero, e integrar Bootstrap (mediante CDN) para cumplir con los requisitos de componentes y clases de utilidad.
